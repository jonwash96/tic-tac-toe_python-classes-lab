# Tic-Tac-Toe & Ultimate Tic-Tac-toe CLI game
## Python classes Lab

**Written in Python**

*Ultimate Tic-Tac-Toe* is a version of the classic Tic-Tac-Toe game where the game is played across 9 sub-boards. The goal is to win the entire game by winning 3 sub-boards in a 3-in-a-row formation. Each turn, the next board played on is determined by the position which corresponds to the previously played cell.
